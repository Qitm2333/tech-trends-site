- **核心模块**: 信息提取引擎、来源定位器、可视化渲染器
- **技术特点**: 与Gemini深度集成、支持多种输出格式
- **创新点**: 精确的来源grounding，可追踪每条提取信息的原文出处

**关键数据**

- Stars: 8.5K (+1.2K 今日)
- 语言: Python (98%)
- 许可证: Apache 2.0

**核心能力**

- 从任意非结构化文本提取结构化数据
- 精确的来源定位和验证
- 交互式可视化展示
- 支持自定义提取模板

**市场影响**

Google通过langextract正式进入LLM工具市场，与LangChain、LlamaIndex等形成直接竞争。其精确的来源定位功能解决了LLM提取信息的可信度问题，对企业级应用具有重要意义。
